<<< Version 2.0.0 >>>

IMPROVEMENTS
* Struct properties can be controlled as multi-single properties in tween components now.
* You can reference materials directly in tween material components, if it's null you can also select a material from a renderer.
* New method of Path: GetClosestSplinePosition
* New method of Path: GetClosestPathPosition
* New method of TweenInterpolator: Create
* New component: TweenMaterialVector
* New component: TweenAnchoredPosition3D
* New component: TweenOffsetMin
* New component: TweenOffsetMax
* New component: TweenLightColor
* New component: TweenLightIntensity

CHANGES
* Remove component: TweenMaterialAlpha (Use TweenMaterialColor instead)
* Remove component: TweenImageAlpha (Use TweenImageColor instead)
* Remove component: TweenTextAlpha (Use TweenTextColor instead)
